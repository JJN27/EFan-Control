What's Here
-----------

After compiling:

- from this folder's context, run the installFanControl.sh script to do an install
on this system ... designed to be used during the "change code, compile, test" cycle

- from this folder's context, run the copyToInstaller.sh script to populate the
Installer folder with the build/release files required to build a new installer
package

After compiling and running the copyToInstall.sh script:

- open the ...FanControl Installer Project.pmdoc and use Xcode's PackageMaker to
build a new installer
